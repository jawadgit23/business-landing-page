# Business Landing Page

A modern, responsive business website built with HTML, CSS, and JavaScript.  
Designed to represent a typical client-facing company site with clean layout, accessibility, and mobile-first responsiveness.

## Live Demo

https://jawad23-portfolio.netlify.app/

## Repository

https://github.com/jawadgit23/business-landing-page

## Features

- Responsive layout (mobile, tablet, desktop)
- Multi-section landing page structure
- Services, About, Contact, Footer sections
- Smooth scrolling navigation
- Scroll-based reveal animations
- Form UI with client-side validation
- Clean visual hierarchy and spacing

## Technology Stack

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6, DOM)
